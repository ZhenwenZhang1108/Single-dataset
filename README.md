# Single-dataset:基于Seurat包的sc全流程分析
 You can also look at: https://satijalab.org/seurat/   
 五大R包，唯Seurat独尊  
 大家需要熟练掌握5个R包，分别是: scater,monocle,Seurat,scran,M3Drop  
## 基本步骤
 step1: 创建对象   
 step2: 质量控制QC   
 step3: 表达量的标准化和归一化  
 step4: 去除干扰因素(多个样本整合)  
 step5: 判断重要的基因  
 step6: 多种降维算法  
 step7: 可视化降维结果  
 step8: 多种聚类算法  
 step9: 聚类后找每个细胞亚群的标志基因  
 step10: 继续分类（case-study）
